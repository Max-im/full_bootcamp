## __SetUp the app__
 - Download this repo:
  `git clone https://github.com/Max-im/full_bootcamp`
 - Go to the project directory:
  `cd <PATH>`
 - Install all App dependencies:
  `npm i`
 

---


## __Run the App__
 - for start the project in develop mode:   
  `npm start / node app.js`
